**Приоритет:** Критический
**Время:** 1 час
**Файлы:**
- `app/services/notifications.py`
- `app/services/announcements.py`

**Шаги:**

2. Обернуть все `bot.send_message()` в try-except
3. Обработать случаи:
   - Пользователь заблокировал бота → пометить пользователя как неактивного
   - Чат не найден → логировать ошибку
   - Другие ошибки API → логировать и продолжать работу
4. Добавить retry логику для временных ошибок
5. Обновить статус пользователя в БД при блокировке

**Код примера:**
```python
try:
    await bot.send_message(...)
except BotBlocked:
    logger.warning(f"User {user_id} blocked the bot")
    # Обновить is_active = False
except ChatNotFound:
    logger.error(f"Chat {chat_id} not found")
except TelegramAPIError as e:
    logger.error(f"Telegram API error: {e}")
```

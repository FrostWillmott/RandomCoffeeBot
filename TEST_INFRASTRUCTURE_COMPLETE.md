## 🚀 Использование

### Простой способ (рекомендуется):

```bash
# Все автоматически
make test
```

### Ручное управление:

```bash
# Запустить тестовую БД
make test-db-up

# Запустить тесты
uv run pytest tests/ -v

# Остановить тестовую БД
make test-db-down
```
